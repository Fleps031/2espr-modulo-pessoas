# Módulo Pessoas

Um módulo Java desenvolvido para gerenciar informações de pessoas.

## 📋 Descrição

O **Módulo Pessoas** é um projeto Java que implementa funcionalidades essenciais para administração e processamento de dados de pessoas. Este módulo fornece uma base sólida para sistemas que necessitam gerenciar registros de usuários, contatos e informações pessoais.

## 🚀 Funcionalidades

- Criação e gerenciamento de registros de pessoas
- Validação de dados pessoais
- Armazenamento e recuperação de informações
- Operações CRUD (Create, Read, Update, Delete)

## 🛠️ Tecnologias

- **Linguagem**: Java
- **IDE**: IntelliJ IDEA (suportado)

```
2espr-modulo-pessoas/

```

## 🔧 Instalação e Configuração

### Pré-requisitos

- Java 8 ou superior
- IntelliJ IDEA (opcional, mas recomendado)
- Git

### Passos para Começar

1. Clone o repositório:
```bash
git clone https://github.com/Fleps031/2espr-modulo-pessoas.git
```

2. Navegue até o diretório do projeto:
```bash
cd 2espr-modulo-pessoas
```

3. Abra o projeto no IntelliJ IDEA ou compile com:
```bash
javac -d bin src/**/*.java
```
